# 42 AI Tutor

Plug-and-play prompt for setting up a 42 C AI tutor in ChatGPT, Gemini, or Perplexity.

## Use It

Open [SETUP.md](SETUP.md), copy the prompt, and paste it into your AI workspace instructions.

Add your current 42 subject PDF and Norm file if you have them.

## What It Does

This tutor helps students reason through C, debug, test, follow the 42 Norm, and prepare for peer evaluation.

It should not give submit-ready 42 project solutions.
