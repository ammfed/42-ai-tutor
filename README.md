# 42 C AI Tutor Setup

Plug-and-play setup for a 42 C tutor in ChatGPT Projects, Gemini Gems, or Perplexity Spaces.

## Use It

1. Create a new AI workspace named `42 C AI Tutor`.
2. Paste [PROMPT.md](PROMPT.md) into the workspace instructions.
3. Upload the files in [knowledge-files/](knowledge-files/).

## Knowledge Files

- [en.norm.pdf](knowledge-files/en.norm.pdf)
- [ai_instructions.pdf](knowledge-files/ai_instructions.pdf)
- [tutor-guide.md](knowledge-files/tutor-guide.md)
- [tutor-behavior.md](knowledge-files/tutor-behavior.md)

This tutor should help students reason, debug, test, follow the 42 Norm, and prepare for peer evaluation without giving submit-ready 42 project solutions.
